# Hotel Booking Cancellation Prediction

## 📌 Project Overview

This project aims to predict whether a hotel booking will be **canceled** or **not canceled** using Machine Learning classification techniques. By analyzing booking details, customer information, and reservation characteristics, the model helps hotels anticipate cancellations and improve operational planning.

---

## 📊 Dataset

The dataset contains booking records from hotels with various customer, reservation, and stay-related features.

### Target Variable

* **is_canceled**

  * **0** → Booking Not Canceled
  * **1** → Booking Canceled

### Features

The dataset includes information such as:

* Hotel Type
* Lead Time
* Arrival Date
* Length of Stay
* Number of Adults, Children, and Babies
* Meal Type
* Country
* Market Segment
* Distribution Channel
* Repeated Guest Status
* Previous Cancellations
* Previous Bookings Not Canceled
* Reserved and Assigned Room Type
* Booking Changes
* Deposit Type
* Customer Type
* Average Daily Rate (ADR)
* Required Car Parking Spaces
* Number of Special Requests
* Reservation Status

---

## 🎯 Objective

Build a classification model that accurately predicts whether a hotel reservation is likely to be canceled based on booking and customer-related features.

---

## 🛠️ Technologies Used

* Python

---

## 📈 Results

The trained classification model learns booking patterns associated with cancellations and can be used to estimate the probability of future reservation cancellations.
